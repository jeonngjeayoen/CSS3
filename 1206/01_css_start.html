01_css소개와 문법

css는 cascading style sheets의 약자로서 html문서의 내용에 서식을 적용하기 위한 언어로서 미국의 '에릭 메이어'라는 개발자에 의해서 개발된 언어이다.

현재까지 css level3까지 나온 상태로서 웹문서의 레이아웃, 그림자효과, 애니메이션 등의 동적인 콘텐츠도 개발이 가능한 상태이다.

css의 특징은 웹문서 제작시 다양한 사용자 환경에서 동일한 내용을 볼 수 있게 표현해주는 언어로서 웹표준 제작시 필수언어이다.

1. css문법

css언어는 기본적으로 head와 head사이에 style선언 또는 link태그를 사용하여 외부서식을 연결 할 수 있다.

선택자{속성:값;속성:값;}

  • 선택자(selector) : 서식을 적용하기 위한 하나의 방법을 말하며 대략 50가지가 있다.
  • 속성(property) : 속성은 서식의 종류로서 예를 들어 글자모양, 글자색상, 배경색, 가로크기, 세로크기, 위치 지정하기, 애니메이션 등의 서식 내용을 말한다.
  • 값(value) : 속성을 선택하면 반드시 설정하는 내용을 말한다. 예를 들면 'font-size:20px;background:#f00'

2. 선택자 학습하기(기본)

1) 클래스 선택자(.class) - 클래스 선택자는 사용자가 지정한 영역에 서식을 적용하기 위한 선택자로서 한번 작성된 선택자는 언제든 다시 사용할 수 있는 서식을 적용할 때 사용한다. class선택자는 사용자가 이름을 작성하여 적용하는데, 이름을 작성할 때 의미있는 이름으로 작성하며 이름앞에 '.'기호를 붙여주는 것이 특징이다.

2) 아이디 선택자(#id) - 아이디 선택자는 클래스 선택자와 마찬가지로 사용자가 지정한 영역에 서식을 적용하기 위한 선택자로서 클래스 선택자와 다른 점은 서식을 반복해서 사용하지 않고, 중복되지 않는 서식을 만들고자 사용한다. 아이디 이름 작성시 이름앞에 '#'기호를 붙인다.

3) 태그 선택자(tag) - 사용자가 지정한 태그를 문서안에서 찾아 서식을 적용하기 위한 선택자이다. 서식이 한꺼번에 변경되기 때문에 실제 사용시 원하지 않는 부분까지도 적용되기 때문에 주의해야 한다.
